# 项目简介

## 核心功能
- 基于 YOLOv8 的工地安全防护衣检测系统
- 支持 5 类目标检测：头盔、无头盔、无背心、人、背心
- 提供完整数据集（1200张图片）和训练好的模型
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户界面，实时显示目标位置、目标总数和置信度
- 支持自定义置信度和 IOU 阈值
- 可选择显示标签和原图
- 支持选择不同检测模型
- 支持批量检测和结果查看

## 适用场景
- 建筑工地安全监控
- 安全防护装备合规性检查
- 工作人员安全状态实时监测
- 安全管理系统集成

## 技术特点
- 基于 Python 3.9 开发
- 使用 OpenCV-Python、PyQt5、Ultralytics 和 Torch1.9 等技术栈
- 提供完整源码、数据集、训练模型和运行教程
- 支持跨平台使用（PyCharm+Anaconda 或 VSCode+Anaconda）
